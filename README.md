# Scikit Learn - Estudos de Machine Learning

## Introdução

### O que é Scikit-Learn

- É a biblioteca mais usada de aprendizado de máquina em Python;
- Oferece ferramentas eficientes para aprendizado de máquina e modelagem estatística
  - classificação
  - regressão
  - agrupamento e redução de dimensionalidade
- Construída sobre NumPy, SciPy e Matplotlib

### Origem do Scikit-Learn

- Desenvolvido inicialemente por David Cournapeau;
- Em 2010, Fabian Pedregosa, Gael Varoquaux, Alexandre Gramfort e Vincent Michel, do FIRCA(Instituto Francês de Pesquisa em Ciência da Computação e Automação), levou este projeto a outro nível.

### Pré-requisitos

- Python >= 3.5
- NumPy >= 1.11.0
- SciPy >= 0.17.0
- Joblib >= 0,11
- Matplotlib >= 1.5.1
- Pandas >= 0.18.0

### Instalação

#### Usando o PIP

`pip install -U scikit-learn`

#### Usando o conda

`conda install scikit-learn`
